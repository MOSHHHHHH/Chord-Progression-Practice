
**תרגול הרמוני - פרויקט תאוריה מוזיקלית
**פרויקט זה הוא אפליקציית רשת פשוטה לתרגול תאוריה מוזיקלית והרמוניה. הוא מאפשר למשתמשים לבחור סולם (מז'ור או מינור) בכל תו שורש, ולחקור שני נושאים עיקריים:

מבנה הסולם: הצגת המרווחים הבונה את הסולם והתווים המרכיבים אותו.

מהלכי אקורדים: הצגת עשרות מהלכי אקורדים נפוצים הרלוונטיים לסולם הנבחר, כולל ניתוח הרמוני של כל אקורד.

הפרויקט נבנה באמצעות HTML, CSS ו-JavaScript (Vanilla JS) בלבד, וכל המידע התאורטי (תווים, אקורדים, סולמות ומהלכים) מוטמע ישירות בקוד.

**✨ תכונות עיקריות
**בחירת סולם דינמית: בחר כל אחד מ-12 תווי השורש, קבע דיאז (♯), במול (♭) או טבעי, ובחר את סוג הסולם (מז'ור / מינור).

תרגול סולמות:
הצגה ברורה של המרווחים (טון, חצי טון וכו') המרכיבים את הסולם.

אפשרות ל"גילוי" התווים המדויקים בסולם.

תרגול מהלכי אקורדים:

ספרייה של 50 מהלכים שונים לסולם מז'ור ו-50 לסולם מינור.

ניווט נוח בין המהלכים (הבא/הקודם).

הצגת המהלך באמצעות ספרות רומיות (לניתוח הרמוני) ושמות האקורדים (לנגינה).

מצב תרגול: אפשרות להסתרת שמות האקורדים (באמצעות מתג) כדי לאתגר את הזיכרון ולתרגל זיהוי לפי ספרות רומיות.

ניתוח אקורדים (מודאל):

לחיצה על כל אקורד במהלך פותחת חלון מידע.

החלון מציג את התווים המרכיבים את האקורד.

הסבר הרמוני על סוג האקורד (למשל, "דומיננט 7", "מינור 7b5").

**🚀 איך משתמשים?
**פתח את הקובץ index.html בדפדפן.

במסך הראשי, בחר תו שורש (לדוגמה 'C'), סימן היתק ('טבעי') וסוג סולם ('מז'ור').

לחץ על "תרגול סולם" כדי לראות את מבנה הסולם, או על "תרגול מהלכי אקורדים" כדי להתחיל בתרגול ההרמוני.

במסך המהלכים, השתמש בכפתורי הניווט כדי לעבור בין המהלכים השונים.

**📖 מה זה "מהלכי אקורדים"?
**מהלך אקורדים (Chord Progression) הוא למעשה רצף של אקורדים המנוגנים בזה אחר זה. זהו אחד המרכיבים הבסיסיים ביותר במוזיקה, שכן הוא יוצר את השלד ההרמוני של השיר וקובע את התחושה הרגשית שלו.

מהלכים הרמוניים פועלים על ידי יצירת מתח ושחרור (Tension and Release). אקורדים מסוימים (כמו הדומיננט, V) יוצרים "מתח" שגורם למאזין לצפות ל"פתרון" חזרה לאקורד ה"בית" (הטוניקה, I).

**ספרות רומיות
**כדי לתאר מהלכי אקורדים בצורה אוניברסלית, אנו משתמשים בספרות רומיות. במקום לומר "בסולם דו מז'ור, נגן C, אחרי זה G, ואז Am...", אנו משתמשים במספרים המייצגים את מיקום האקורד בתוך הסולם.

כל דרגה בסולם (מ-1 עד 7) מקבלת ספרה רומית.

אותיות גדולות (I, IV, V) מייצגות אקורדים מז'וריים.

אותיות קטנות (ii, iii, vi) מייצגות אקורדים מינוריים.

סימן מוקטן (vii°) מייצג אקורד מוקטן (Diminished).

לדוגמה, בסולם דו מז'ור (C Major), האקורדים הדיאטוניים (הטבעיים לסולם) הם:

I - דו מז'ור (C)

ii - רה מינור (Dm)

iii - מי מינור (Em)

IV - פה מז'ור (F)

V - סול מז'ור (G)

vi - לה מינור (Am)

vii° - סי מוקטן (Bdim)

לכן, "מהלך ארבעת האקורדים" המפורסם (I-V-vi-IV) יתורגם בסולם זה ל: C - G - Am - F.

האפליקציה הזו מאפשרת לך לתרגל זיהוי של המהלכים הללו בכל סולם שתבחר.

**🛠️ טכנולוגיה
**הפרויקט נבנה בכוונה תוך שימוש בטכנולוגיות בסיס בלבד, ללא ספריות חיצוניות או פריימוורקס (מלבד פונט גוגל):

HTML5: מבנה סמנטי של האפליקציה.

CSS3: עיצוב מודרני, כולל שימוש במשתני CSS (Custom Properties) לניהול צבעים ועיצוב.

JavaScript (ES6+): כל הלוגיקה של האפליקציה, כולל ניהול מצב, יצירת אלמנטים דינמית, וכל בסיס הנתונים התאורטי.
